# 조족등 - AI 기반 스마트 농촌 조명 솔루션

> **"옛날 밤길을 걸을 때 발 밑을 비추기 위해 사용하던 등불, 현대 기술로 다시 태어나다."**

## 프로젝트 개요
**조족등**은 가로등이 부족한 농촌 지역의 어두운 도로 환경에서 보행자의 안전을 지키기 위해 고안된 **AI 기반 스마트 조명 솔루션**입니다.
단순히 길을 비추는 것을 넘어, 날씨와 환경을 인식하고 해충을 유인하는 등 농촌 생활환경에 특화된 기능을 제공합니다.

* **팀명:** 뚜알기농장
* **주제:** 지속 가능한 농촌을 위한 기술 (Safety & Eco-friendly)

## 문제 인식 (Background)
* **위험한 밤길:** 가로등이 드문 시골 버스 정류장 및 도로변의 시야 확보 어려움.
* **보행자 사고:** 차량 운전자가 어둠 속의 보행자를 식별하지 못해 발생하는 사고 위험.
* **비효율적 조명:** 불필요하게 낭비되는 전력 및 기존 가로등의 빛 공해.

## 핵심 기능 및 기술 (Key Features)

### 1. 상황별 스마트 조명 시스템
| 조명 모드 | 색상 | 작동 조건 및 효과 |
| :--- | :--- | :--- |
| **백색등 (White)** | ⚪ | 맑은 날씨에 작동하여 밝고 선명한 시야 제공 |
| **황색등 (Yellow)** | 🟡 | 비나 안개가 낀 날 작동하여 높은 투과율로 가시성 확보 |
| **UV 램프** | 🟣 | 파장이 짧은 청색광을 도로 바깥쪽으로 비춰 **벌레를 유인**, 보행자 쾌적성 증대 |

### 2. 다중 센서 융합 (Sensor Fusion)
* **압력 센서:** 발자국(보행)을 인식하여 사람이 지나갈 때만 점등 (에너지 절약).
* **라이다 (LiDAR):** 사물의 형태와 거리를 인식하여 사람, 차량, 야생동물을 구별.
* **우적 센서:** 비가 오는지 감지하여 자동으로 조명 색상을 황색으로 변경.
* **조도 센서:** 주변 밝기를 감지하여 주간/야간 모드 자동 전환.

## 프로토타입 진화 과정 (Prototype Evolution)
최적의 솔루션을 찾기 위해 4단계의 프로토타입을 비교 분석했습니다.

| 모델명 | 특징 | 장점 | 단점 |
| :--- | :--- | :--- | :--- |
| **1. 나무애** | 가로수 부착형 | 별도 구조물 불필요 | 나무 위치에 의존, 조명/UV 동시 사용 불가 |
| **2. 바닥매립등** | 지면 매립형 | 깔끔한 미관 | 눈부심 발생, 보행 걸림 사고 위험 |
| **3. 조족등 (Final)** | **보행자 맞춤형** | **높은 시인성, 벌레 유인 분리 설계** | 설치 구조물 필요 |
| **4. 스마트가로등** | 기존 가로등 형태 | 넓은 범위 커버 | 높은 설치 비용, 빛 공해 우려 |

## 웹사이트 소개 (Web Preview)
이 저장소는 아이디어 해커톤의 결과물을 시각적으로 보여주는 **소개 웹페이지**를 포함합니다.
* **Design Concept:** 어두운 밤길을 비추는 조족등의 이미지를 형상화한 **Dark Mode UI**.
* **Language:** HTML5, CSS3.

## 기대 효과
1. **안전 증진:** 농촌 고령층의 낙상 사고 예방 및 야간 보행자 교통사고 감소.
2. **에너지 효율:** 센서 기반 작동으로 불필요한 전력 소모 최소화.
3. **생활 편의:** 해충 유인 기능을 통해 쾌적한 야간 보행 환경 조성.
